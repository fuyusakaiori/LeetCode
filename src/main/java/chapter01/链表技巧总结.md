#1 快慢指针
```java
// 这里返回的是第二个中点
public ListNode findPoint(ListNode head){
    ListNode fast = head;
    ListNode slow = head;
    while(fast != null && fast.next != null){
        fast = fast.next.next;
        slow = slow.next;
    }
    return slow;
}
```

#2 反转链表
```java
public ListNode reverse(ListNode head){
    ListNode previous = null;
    ListNode current = head;
    ListNode next = null;
    while(current != null){
        next = current.next;
        current.next = previous;
        previous = current;
        current = next;
    }
    return previous;
}
```
